# vw与%的区别

---

## 面试题：vw 单位和 % 单位有什么区别？

**答案：** vw 与视口有关，% 取决于父元素的宽度，有继承情况

**详细说明：**

### 1. vw 单位（Viewport Width）

**定义：** vw 是相对于视口宽度的单位

```css
/* 1vw = 视口宽度的 1% */
.element {
  width: 50vw;  /* 视口宽度的一半 */
  font-size: 4vw; /* 字体大小随视口变化 */
}
```

**特点：**

- **相对于视口**：始终基于浏览器窗口的宽度
- **响应式设计**：适合做响应式布局
- **不受父元素影响**：独立于父元素尺寸
- **视口变化时自动调整**：窗口大小改变时自动重新计算

**实际应用：**

```css
/* 全屏横幅 */
.banner {
  width: 100vw;
  height: 30vw;
  background: linear-gradient(to right, #667eea, #764ba2);
}

/* 响应式字体 */
.responsive-text {
  font-size: 3vw;  /* 在手机上小，在大屏上大 */
}
```

### 2. % 单位（百分比）

**定义：** % 是相对于父元素的单位

```css
/* 子元素的 % 是相对于父元素的 */
.parent {
  width: 600px;
}

.child {
  width: 50%;  /* 600px * 50% = 300px */
  padding: 10%;  /* 600px * 10% = 60px */
}
```

**特点：**

- **相对于父元素**：基于父元素的尺寸计算
- **继承性**：父元素改变，子元素自动调整
- **不同属性参考不同**：
  - width/height/padding/margin：相对于父元素的对应属性
  - font-size：相对于父元素的字体大小
  - line-height：相对于父元素的字体大小

**实际应用：**

```css
/* 流式布局 */
.container {
  width: 100%;
  max-width: 1200px;
}

/* 相对父元素 */
.sidebar {
  width: 25%;  /* 占据父元素宽度的 1/4 */
}
```

### 3. 继承情况详解

**% 的继承性：**

```css
/* 父元素设置字体大小 */
.parent {
  font-size: 20px;
}

/* 子元素的 % 相对于父元素的 font-size */
.child {
  font-size: 150%;  /* 20px * 150% = 30px */
  line-height: 120%;  /* 30px * 120% = 36px */
}
```

**vw 无继承性：**

```css
/* vw 始终相对于视口，不受父元素影响 */
.parent {
  width: 800px;
}

.child {
  width: 50vw;  /* 视口宽度的一半，与父元素无关 */
}
```

### 4. 对比总结表

| 特性 | vw | % |
|------|-----|-----|
| 参考对象 | 视口宽度 | 父元素 |
| 继承性 | 无 | 有（相对于父元素） |
| 响应式 | 天然响应式 | 需要配合媒体查询 |
| 适用场景 | 全屏元素、响应式字体 | 流式布局、相对尺寸 |
| 计算方式 | 视口宽度 × 百分比 | 父元素属性 × 百分比 |

### 5. 实际应用示例

**场景 1：响应式字体（推荐 vw）**

```css
/* 使用 vw 实现响应式字体 */
h1 {
  font-size: 5vw;  /* 大标题 */
}

p {
  font-size: 2.5vw;  /* 正文 */
}

/* 在小屏幕上自动缩小，大屏幕上自动放大 */
```

**场景 2：流式布局（推荐 %）**

```css
/* 使用 % 实现流式布局 */
.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
}

.sidebar {
  width: 30%;
}

.main {
  width: 70%;
}
```

**场景 3：混合使用**

```css
/* 宽度用 %，字体用 vw */
.card {
  width: 80%;
  padding: 2vw;
  font-size: 1.5vw;
}
```
