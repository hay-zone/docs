# padding与margin的区别

---

## 面试题：padding 和 margin 有什么区别？

**答案：** 作用对象不同，前者作用于自身，后者作用于外部对象

**详细说明：**

### 1. 作用对象不同

**Padding（内边距）：**

- **作用于元素自身**：在元素内容与边框之间创建空间
- **增加元素尺寸**：padding 会增加元素的实际尺寸（除非设置 `box-sizing: border-box`）
- **影响背景色**：背景色会延伸到 padding 区域

**Margin（外边距）：**

- **作用于外部对象**：在元素边框与相邻元素之间创建空间
- **不增加元素尺寸**：margin 不影响元素本身的尺寸
- **影响布局流**：margin 会影响元素在文档流中的位置

### 2. 代码示例对比

```css
.box {
  width: 200px;
  height: 200px;
  background: lightblue;
  padding: 20px;  /* 内边距 */
  margin: 30px;    /* 外边距 */
}
```

**实际效果：**

- **元素实际宽度**：200px（内容）+ 40px（左右 padding）= 240px
- **元素实际高度**：200px（内容）+ 40px（上下 padding）= 240px
- **元素与外部间距**：上下左右各 30px
- **背景色范围**：延伸到 padding 区域，不包括 margin 区域

### 3. 视觉对比图

```
┌─────────────────────────────────┐ ← 外部其他元素
│         margin: 30px         │
│  ┌───────────────────────┐  │
│  │  padding: 20px       │  │
│  │ ┌─────────────────┐   │  │
│  │ │   内容区域     │   │  │
│  │ │   200x200px    │   │  │
│  │ └─────────────────┘   │  │
│  └───────────────────────┘  │
│         margin: 30px         │
└─────────────────────────────────┘ ← 外部其他元素
```

### 4. 特殊情况

**Margin 塌陷（Margin Collapse）：**

- 相邻垂直方向的 margin 会合并
- padding 不会发生塌陷

**示例：**

```css
.box1 {
  margin-bottom: 20px;
}

.box2 {
  margin-top: 30px;
}

/* 实际间距不是 50px，而是 max(20px, 30px) = 30px */
```

**总结：**

| 特性 | Padding | Margin |
|------|---------|--------|
| 作用位置 | 元素内部 | 元素外部 |
| 影响尺寸 | 是（除非 border-box） | 否 |
| 背景色 | 延伸 | 不延伸 |
| 塌陷 | 不发生 | 垂直方向会塌陷 |
| 负值 | 允许 | 允许 |
